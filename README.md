# 📊 UuAP Case Study 2: Airbnb Pricing Patterns in Mediterranean Cities

This repository contains the final version of **Case Study 2**, focused on analyzing Airbnb pricing patterns across six Mediterranean and Southern European cities: Athens, Malaga, Istanbul, Naples, Rome and Valencia.

The project explores how **location**, **accommodation type**, **host characteristics** and **reviews** influence prices and market behavior in the short-term rental sector.

---

## 🎯 Project Objective

The main research question is:

**Which factors most strongly influence Airbnb prices and do those factors behave differently across cities?**

The analysis focuses on:
- price distributions by city
- the effect of location on pricing
- comparisons between accommodation types
- host and review characteristics
- hypothesis testing and advanced statistical analysis

## 📊 Methodology

The analysis was carried out through several steps:
- data collection and cleaning
- descriptive statistics
- geospatial analysis and mapping
- accommodation type analysis
- host analysis
- review and rating analysis
- regression and correlation analysis
- clustering and advanced exploratory analysis

The project uses Python-based data processing, statistical analysis, visualization and interactive mapping tools.

---

## 🔎 Key Findings

The results show that:
- **Rome** has the most expensive market in the sample
- **Athens** and **Naples** are more affordable
- **accommodation type** is a stronger pricing factor than location in most cities
- **entire homes/apartments** consistently cost more than private rooms
- the relationship between location and price is not the same across all cities
- the advanced analysis reveals useful market outliers, amenity effects and quality patterns
